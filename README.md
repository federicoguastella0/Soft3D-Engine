Un mini-motore di rendering 3D software scritto in C e SDL2. 
Calcola le proiezioni prospettiche e le rotazioni matematiche direttamente sulla CPU, scrivendo i pixel nel framebuffer.

## Funzionalità attuali
- Generazione di forme geometriche (nuvole di punti / sinusoidi).
- Rotazione 3D in tempo reale sull'asse Y.
- Proiezione prospettica corretta.

## Requisiti
- Compilatore C (GCC / Clang / MSVC)
- Librerie SDL2
